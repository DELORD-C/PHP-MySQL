Créer la table `utilisateur` avec les champs suivants :
- id (int) auto incrementation
- nom (varchar - 255)
- email (varchar - 255)
- password (varchar - 255)

- id (int) auto incrementation
- id_film (int)
- id_utilisateur (int)
- texte (varchar 500)
- note (int)
